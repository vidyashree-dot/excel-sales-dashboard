# Excel Sales Dashboard

## Project Overview
This project analyzes sales data using Microsoft Excel. The goal is to understand sales performance across different product categories and regions.

## Dataset
- Sample Superstore dataset
- Contains sales information by Category and Region

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts

## Key Insights
- West region shows the highest sales across categories
- Technology category contributes the most to total sales
- Office Supplies have consistent performance across regions
